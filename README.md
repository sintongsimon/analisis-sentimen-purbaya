# Sentiment Analysis of Public Opinion on Purbaya on Social Media X

This repository contains the dataset and source code used in the research:

"Analisis Sentimen Opini Publik terhadap Tokoh Purbaya di Media Sosial X Menggunakan Support Vector Machine dan IndoBERTweet"

## Dataset

The dataset consists of 1,476 Indonesian tweets related to the public figure Purbaya collected from social media X.

Dataset processing stages:
1. Tweet crawling
2. Dataset merging
3. Data selection
4. Text preprocessing
5. Manual sentiment labeling
6. Model training

## Models Used

- Support Vector Machine (SVM) with TF-IDF
- IndoBERTweet Transformer Model

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Application

The sentiment analysis application can be accessed at:

https://sintongsimon.streamlit.app/

## Repository Structure

- dataset : tweet datasets used in the research
- scripts : data processing and model training scripts
- results : evaluation results
- app : Streamlit sentiment analysis application
